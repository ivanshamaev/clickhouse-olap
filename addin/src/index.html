ClickHouse OLAP

Pivot analysis of ClickHouse data directly in Microsoft Excel.
No raw data leaves ClickHouse — only aggregated results.

  1. Download manifest.xml from the latest release below
  2. In Excel: Insert → Add-ins → My Add-ins → Upload Add-in
  3. Select the downloaded manifest.xml
  4. Click the ClickHouse OLAP button on the Home tab
